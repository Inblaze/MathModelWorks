# 线性规划 - 人员问题

## 题目描述

某厂每日8小时的产量不低于1800件。为了进行质量控制， 计划聘请两种不同水平的检验员。一级检验员的标准为：速度25件/小时，正确率98%，计时工资4元/小时；二级检验员的标准为：速度15件/小时，正确率95%，计时工资3元/小时。 检验员每错检一次，工厂要损失2元。为使总检验费用最省， 该工厂应聘一级、二级检验员各几名？

## 解答

### 假设

设x,y分别表示聘请的一级、二级检验员的数量；

错检件数使用期望值代替。

### 模型建立

我们的目标函数是总检验费用W(x,y):
$$
\begin{align}
W(x,y)
&=8\times 4x+8\times 3y+2[(1-98\%)\times 8\times 25x+(1-95\%)\times 8\times 15y]\\
&=40x+36y
\end{align}
$$
要求W(x,y)最小

加上约束条件如下：
$$
\begin{align}
min\ \  
&W(x,y)=40x+36y\\
s.t\ \ \ \ \ \  
&8\times25x+8\times 15y\geq 1800\\
&x,y\in N
\end{align}
$$

### 程序求解

我们使用Lingo求解，得到的解如下：

最小总检验费用为360.0，在x=9,y=0时取得，即$min\ W=W(9,0)=360.0$

也就是说，为使总检验费用最低，应当聘请9名一级检验员，0名二级检验员，此时总检验费用为360.0元。
